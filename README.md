```
usage: wordlist-combinator.py [-h] [-wc [WORD_COUNT]] [-s [N]] file

Combine a list of words in a list of combined words with separators

positional arguments:
  file

optional arguments:
  -h, --help            show this help message and exit
  -wc [WORD_COUNT], --word-count [WORD_COUNT]
                        How many words to combine together. Default is 3
  -s [N], --separators [N]
                        Separators between combined words. Default is "-_". Empty separator is always included.
```
